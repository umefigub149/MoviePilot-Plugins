# 联动115追更

检测 `115网盘订阅追更-自用版` 与 `115网盘STRM助手` 的成功转存信号，延迟将用户手动配置的固定目录加入 MoviePilot 整理队列。

## 功能

- 轮询 `DownloadHistory.username = P115StrgmSub` 的新增记录。
- 桥接 `P115StrmHelper` 的分享转存成功回调。
- 支持 `CloudDrive储存`、`u115`、`115网盘Plus`、`local` 或自定义目标存储。
- 支持延迟入队、防抖、路径前缀白名单、演练模式。
- 支持追更目录/分享目录解析测试接口。

## 演练模式

演练模式就是“只试跑不真执行”。开启后插件会检测记录、判断目录、写日志，但不会真正把目录加入 MP 整理队列。

建议首次安装时先打开演练模式，确认日志中的目录正确后再关闭。

## 推荐配置

```text
目标存储：CloudDrive储存
追更入队目录：
/网盘整理/网盘待整理目录/Movie
/网盘整理/网盘待整理目录/TV

STRM助手分享转存入队目录：
/最近接收
/网盘整理/分享转存目录

允许入队路径前缀：
/网盘整理
/最近接收
/我的接收
```

## 目标存储说明

- `CloudDrive储存`：适合 MP CloudDrive2 储存插件，填写 115 内部路径，如 `/网盘整理/...`。
- `u115`：适合 MP 的 u115 存储模块。
- `115网盘Plus`：适合 115strm助手增强存储模块。
- `local`：适合容器本地挂载路径。
- 其他名称：按 MP 实际储存名称手动填写。

## 与 115strm助手网盘管理的区别

115strm助手网盘管理是重型功能，会管理 115 文件、STRM、缓存、生命周期、整理接管等。

本插件不是第二个 115strm助手，只做：

```text
成功转存信号 → 延迟 N 秒 → 固定目录加入 MP 整理队列
```

本插件不做：

- 不精确识别本次转存的子目录。
- 不扫描 115 网盘目录。
- 不做目录 diff。
- 不解析分享链接文件名。
- 不读取、不保存、不输出任何 Cookie 或 Token。
- 不接管 MP 整理任务。
- 不处理字幕/音轨关联。

## 工作流

```text
P115StrgmSub 成功转存写入 DownloadHistory
或
P115StrmHelper 分享转存成功
→ 延迟 N 秒
→ 将固定目录加入 MP 整理队列
```
