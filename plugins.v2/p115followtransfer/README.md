# 联动115追更

检测 `115网盘订阅追更-自用版` 与 `115网盘STRM助手` 的成功转存信号，延迟将用户手动配置的固定目录加入 MoviePilot 整理队列。

## 功能

- 轮询 `DownloadHistory.username = P115StrgmSub` 的新增记录。
- 桥接 `P115StrmHelper` 的分享转存成功回调。
- 支持 `CloudDrive储存` 作为目标存储，适合填写 115 网盘内部路径。
- 支持延迟入队、防抖、路径前缀白名单、Dry Run。

## 设计边界

本插件刻意保持简单：

- 不精确识别本次转存的子目录。
- 不扫描 115 网盘目录。
- 不做目录 diff。
- 不解析分享链接文件名。
- 不读取、不保存、不输出任何 Cookie 或 Token。

## 推荐配置

```text
目标存储：CloudDrive储存
追更入队目录：
/网盘整理/网盘待整理目录/Movie
/网盘整理/网盘待整理目录/TV

STRM助手分享转存入队目录：
/最近接收
/网盘整理/分享转存目录

允许入队路径前缀：
/网盘整理
/最近接收
/我的接收
```

## 工作流

```text
P115StrgmSub 成功转存写入 DownloadHistory
或
P115StrmHelper 分享转存成功
→ 延迟 N 秒
→ 将固定目录加入 MP 整理队列
```
